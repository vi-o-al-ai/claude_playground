# Zombie Lane Runner

3D endless lane runner with shooting, built in Godot 4.6.

## Structure

- `scripts/` -- GDScript source files.
- `scenes/` -- Godot scene files (.tscn).
- `assets/` -- 3D models, textures, audio.
- `tests/` -- GUT test files (test_player_model.gd, test_shooter.gd, etc.).

## Config

- `project.godot` -- Godot project config.
- `.gutconfig.json` -- GUT test framework config.
- `export_presets.cfg` -- export settings (Web target for deployment).

## Running Locally

Open `project.godot` in Godot 4.6 editor.
