# Crowd Runner

3D crowd-runner ("Count Masters" style) built in Godot 4.6. Steer a crowd down a road, pass through gates that change the count, then clash with an enemy crowd — larger side wins.

## Structure

- `scripts/` -- GDScript source files.
- `scenes/` -- Godot scene files (.tscn).
- `tests/` -- GUT test files.

## Config

- `project.godot` -- Godot project config.
- `.gutconfig.json` -- GUT test framework config.
- `export_presets.cfg` -- export settings (Web target for deployment).

## Running Locally

Open `project.godot` in Godot 4.6 editor.
